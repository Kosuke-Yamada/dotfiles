# eza コマンド

`.aliases` で定義されている eza（モダンな ls 代替）のエイリアスです。

eza がインストールされている場合のみ有効になります。

## エイリアス一覧

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `ei` | `eza --icons --git` | アイコン + Git ステータス付き一覧 |
| `ea` | `eza -la --icons --git` | 全ファイル（隠しファイル含む）+ 詳細表示 |
| `ee` | `eza -aahl --icons --git` | 全ファイル + 詳細表示（ヘッダー付き） |
| `et` | `eza -T -L 3 -a -I '...' --icons` | ツリー表示（3階層、除外パターン付き） |
| `ls` | `ei` | 基本表示（eza で上書き） |
| `la` | `ea` | 全ファイル表示 |
| `ll` | `ee` | 詳細表示 |
| `tree` | `eza -T --icons` | ツリー表示 |

## 使用例

### 基本的なファイル一覧

```bash
ls      # アイコン + Git ステータス付きで表示
la      # 隠しファイルも含めて表示
ll      # 詳細情報（パーミッション、サイズ等）も表示
```

### ツリー表示

```bash
tree           # 現在のディレクトリをツリー表示
et             # 3階層まで、node_modules/.git/.cache を除外して表示
```

## オプション説明

| オプション | 説明 |
|------------|------|
| `--icons` | ファイルタイプに応じたアイコンを表示 |
| `--git` | Git の状態（変更、ステージ等）を表示 |
| `-l` | 詳細表示（long format） |
| `-a` | 隠しファイル（.で始まるファイル）も表示 |
| `-h` | ヘッダー行を表示 |
| `-T` | ツリー表示 |
| `-L N` | ツリーの深さを N 階層に制限 |
| `-I 'pattern'` | 指定パターンを除外 |

## 注意事項

- eza がインストールされていない場合、標準の `ls --color` が使用されます
- アイコンを正しく表示するには Nerd Font が必要です（HackGen Nerd が Brewfile に含まれています）
