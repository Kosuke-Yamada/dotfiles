# エイリアス一覧

`.aliases` で定義されているコマンドエイリアスの一覧です。

## 履歴

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `history` | `fc -liD 1` | 履歴を日時付きで表示 |
| `hist` | `fc -liD 1` | history の短縮形 |

## 基本コマンド

安全のため上書き・削除前に確認を行うオプション付きです。

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `cd` | `nocorrect cd` | スペル修正を無効化 |
| `cp` | `nocorrect cp -i` | 上書き前に確認 |
| `mv` | `nocorrect mv -i` | 上書き前に確認 |
| `rm` | `nocorrect rm -i` | 削除前に確認 |
| `ln` | `nocorrect ln -i` | 上書き前に確認 |
| `mkdir` | `nocorrect mkdir -p` | 親ディレクトリも作成 |
| `gcc` | `nocorrect gcc` | スペル修正を無効化 |
| `man` | `nocorrect man` | スペル修正を無効化 |
| `ls` | `nocorrect ls --color` | カラー表示（eza がある場合は上書き） |
| `sl` | `ls` | typo 対策 |

## モダン CLI 代替

インストールされている場合のみ有効になります。

| エイリアス | 代替ツール | 説明 |
|------------|------------|------|
| `top` | htop | プロセスモニタ |
| `grep` | ripgrep (rg) | 高速 grep |
| `wc` | cw | 高速 wc |
| `find` | fd | 高速 find |
| `cat` | bat | シンタックスハイライト付き cat |

## ディレクトリ移動

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `..` | `cd ..` | 1つ上へ |
| `...` | `cd ../..` | 2つ上へ |
| `....` | `cd ../../..` | 3つ上へ |
| `.....` | `cd ../../../..` | 4つ上へ |
| `......` | `cd ../../../../..` | 5つ上へ |
| `.......` | `cd ../../../../../..` | 6つ上へ |

## Zellij

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `ze` | `zellij` | zellij 起動 |
| `zes` | `zellij -s` | 新規セッション作成 |
| `zea` | `zellij a` | セッション再接続 |
| `zed` | `zellij delete-session --force` | セッション強制削除 |
| `zels` | `zellij ls` | セッション一覧 |
