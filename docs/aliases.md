# エイリアス一覧

`.aliases` で定義されているコマンドエイリアスの一覧です。

## 履歴

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `history` | `fc -liD 1` | 履歴を日時付きで表示 |
| `hist` | `fc -liD 1` | history の短縮形 |

## 基本コマンド

安全のため上書き・削除前に確認を行うオプション付きです。

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `cd` | `nocorrect cd` | スペル修正を無効化 |
| `cp` | `nocorrect cp -i` | 上書き前に確認 |
| `mv` | `nocorrect mv -i` | 上書き前に確認 |
| `rm` | `nocorrect rm -i` | 削除前に確認 |
| `ln` | `nocorrect ln -i` | 上書き前に確認 |
| `mkdir` | `nocorrect mkdir -p` | 親ディレクトリも作成 |
| `gcc` | `nocorrect gcc` | スペル修正を無効化 |
| `man` | `nocorrect man` | スペル修正を無効化 |
| `sl` | `ls` | typo 対策 |

## モダン CLI 代替

インストールされている場合のみ有効になります。

| エイリアス | 代替ツール | 説明 |
|------------|------------|------|
| `top` | htop | プロセスモニタ |
| `grep` | ripgrep (rg) | 高速 grep |
| `wc` | cw | 高速 wc |
| `find` | fd | 高速 find |
| `cat` | bat | シンタックスハイライト付き cat |

## eza（モダンな ls 代替）

eza がインストールされている場合のみ有効になります。

| エイリアス | 展開後 | 説明 |
|------------|--------|------|
| `ei` | `eza --icons --git` | アイコン + Git ステータス付き一覧 |
| `ea` | `eza -la --icons --git` | 全ファイル（隠しファイル含む）+ 詳細表示 |
| `ee` | `eza -aahl --icons --git` | 全ファイル + 詳細表示（ヘッダー付き） |
| `et` | `eza -T -L 3 -a -I '...' --icons` | ツリー表示（3階層、除外パターン付き） |
| `ls` | `ei` | 基本表示（eza で上書き） |
| `la` | `ea` | 全ファイル表示 |
| `ll` | `ee` | 詳細表示 |
| `tree` | `eza -T --icons` | ツリー表示 |

### eza オプション説明

| オプション | 説明 |
|------------|------|
| `--icons` | ファイルタイプに応じたアイコンを表示 |
| `--git` | Git の状態（変更、ステージ等）を表示 |
| `-l` | 詳細表示（long format） |
| `-a` | 隠しファイル（.で始まるファイル）も表示 |
| `-h` | ヘッダー行を表示 |
| `-T` | ツリー表示 |
| `-L N` | ツリーの深さを N 階層に制限 |
| `-I 'pattern'` | 指定パターンを除外 |

### 注意事項

- eza がインストールされていない場合、標準の `ls --color` が使用されます
- アイコンを正しく表示するには Nerd Font が必要です
