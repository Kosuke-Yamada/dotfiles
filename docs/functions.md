# カスタム関数一覧

`.functions` で定義されているシェル関数の一覧です。

## ghq リポジトリ検索

### `ghq_repo_search_with_peco`

ghq で管理しているリポジトリをインタラクティブに検索して移動します。

| キーバインド | 説明 |
|--------------|------|
| `Ctrl+]` | ghq リポジトリを peco で検索 |

**動作:**
1. `ghq list -p` で全リポジトリのフルパスを取得
2. peco でインタラクティブにフィルタリング
3. 選択したディレクトリに `cd`

**使用例:**
```bash
# Ctrl+] を押す
# "dotfiles" と入力して絞り込み
# Enter で選択したリポジトリに移動
```

## コマンド履歴検索

### `history_search_with_peco`

コマンド履歴をインタラクティブに検索します。

| キーバインド | 説明 |
|--------------|------|
| `Ctrl+R` | コマンド履歴を peco で検索 |

**動作:**
1. `history -n 1` で履歴を取得
2. `tac` で逆順（新しいものが上）に
3. `awk '!a[$0]++'` で重複を除去
4. peco でインタラクティブに選択
5. 選択したコマンドをコマンドラインに挿入

**使用例:**
```bash
# Ctrl+R を押す
# "git" と入力して git 関連のコマンドを絞り込み
# Enter で選択したコマンドをコマンドラインに挿入
```

## Zellij セッション補完

### `_zellij_sessions`

`zea`（attach）と `zed`（delete）エイリアス用のセッション名補完を提供します。

**動作:**
- `zellij list-sessions` から既存セッション一覧を取得
- Tab キーで補完候補として表示

**使用例:**
```bash
zea <Tab>  # 既存セッション名が補完候補として表示される
zed <Tab>  # 削除対象のセッション名が補完候補として表示される
```
