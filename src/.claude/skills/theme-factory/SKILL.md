# Theme Factory

スライド、ドキュメント、ランディングページに一貫したプロフェッショナルなテーマを適用するツールキット。

## 利用可能なテーマ

| テーマ名 | 特徴 |
|---------|------|
| Ocean Depths | 深海の青をベースにした落ち着いたトーン |
| Sunset Boulevard | 夕焼けの暖色系グラデーション |
| Forest Canopy | 森林の緑をベースにした自然派 |
| Modern Minimalist | シンプルでクリーンなモノトーン |
| Golden Hour | ゴールデンアワーの温かみ |
| Arctic Frost | 氷のような涼しげなブルー |
| Desert Rose | 砂漠のピンクとベージュ |
| Tech Innovation | 先進的なテックブルー |
| Botanical Garden | 植物園の緑と花の色 |
| Midnight Galaxy | 夜空と銀河のダーク系 |

## ワークフロー

1. **テーマ選択**: 上記のプリセットから選択、またはカスタムテーマを作成
2. **確認**: 選択したテーマのカラーパレットとフォントペアリングを確認
3. **適用**: アーティファクト全体にテーマを適用

## カスタムテーマ

プリセットが合わない場合、以下の情報を提供：
- 希望する雰囲気・トーン
- ターゲットオーディエンス
- 参考にしたいブランドやデザイン
- 使用コンテキスト（プレゼン、ウェブ、ドキュメントなど）

## テーマ仕様

各テーマには以下が含まれる：
- Hexカラーコード（プライマリ、セカンダリ、アクセント、背景、テキスト）
- 推奨フォントペアリング（見出し、本文）
- 適用ガイドライン
